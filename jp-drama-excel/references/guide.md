# 일본어 드라마 문장 분석 작업 가이드

## 엑셀 열 구조

| 열 | 내용 | 설명 |
|----|------|------|
| A | 문장 번호 | 원본 유지 |
| B | 단어/문법 분석 | Claude가 채움 |
| C | 일본어 원문 | 원본 B열 이동, 절대 변경 금지 |
| D | 한국어 번역 | Claude가 채움 |

## B열 작성 규칙 (단어/문법 분석)

- 모든 단어 + 문법 패턴 추출 (명사, 동사, 형용사, 조사 포함)
- 최소 2개 이상 항목
- 동사는 반드시 원형(辞書形)으로 표기
- 형식: `単語(よみがな) - 한국어의미, ~文法 - 문법의미`
- 등장인물명(사람 이름)은 제외

**예시:**
```
どこ - 어디, 行く(いく) - 가다, ~たって - ~해도, 税理士(ぜいりし) - 세무사, ~は - ~는, 続ける(つづける) - 계속하다, ~ていける - ~해 나갈 수 있다, ~んだろうけど - ~겠지만
```

## D열 작성 규칙 (한국어 번역)

- 자연스러운 구어체 번역 (드라마 대사)
- 문장 미완성 시 `…` 사용
- 인물명은 괄호로 표시: `(코조)`, `(킨조)`, `(나레이션)`
- 직역보다 자연스러운 의역 우선

## 예시

| A | B | C | D |
|---|---|---|---|
| 4 | 気持ち(きもち) - 마음, 入れ直す(いれなおす) - 다시 가다듬다, ~て - ~해서 | 気持ち 入れ直して | 마음 다시 가다듬고 |
| 8 | 社長(しゃちょう) - 사장, ~が - ~가, 一代(いちだい) - 한 세대, 業界(ぎょうかい) - 업계, 大手(おおて) - 대기업, ~にまで - ~까지, 築き上げる(きずきあげる) - 쌓아올리다, ~られた - 존경표현 | 山王耕造社長が一代で業界大手にまで築き上げられた⸺ | 산노 코조 사장님이 한 대로 업계 대기업까지 쌓아올리셨습니다 |
| 53 | もう - 이미, 焼ける(やける) - 타다, ~てますよ - ~고 있습니다 | （金城）もう焼けてますよ | (킨조) 이미 다 구워졌어요 |

## 작업 금지 사항

- ❌ C열 내용 변경
- ❌ 동사를 활용형으로 표기 (반드시 원형)
- ❌ B열에 항목 1개만 추출
- ❌ 직역만 고집

## 완성 기준 체크리스트

- [ ] C열 원문이 원본과 일치
- [ ] B열 최소 2개 이상 항목
- [ ] 동사 원형 표기 + 히라가나 읽기
- [ ] D열 자연스러운 구어체 번역
- [ ] 인물명 괄호 처리
