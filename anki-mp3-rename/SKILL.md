# Anki MP3 이름 변경

## 개요

Anki CSV에서 Cloze 카드의 일본어 텍스트를 추출해 숫자로만 된 MP3 파일을 `순번_일본어텍스트.mp3` 형식으로 일괄 변경한다.

## 워크플로

1. 사용자에게 MP3 폴더 경로와 CSV 파일 경로 확인
2. `scripts/rename_mp3.py` 실행:
   ```
   python rename_mp3.py "<mp3_폴더>" "<csv_파일>"
   ```
3. 결과 보고

## 스크립트 동작 방식

- **CSV 스트리밍 파싱**: `[sound:` 포함 줄만 처리 (대용량 파일 최적화)
- **추출 패턴**:
  - MP3 파일명: `\[sound:(\d+\.mp3)\]`
  - Cloze 내용: `\{\{c1::(.*?)::`
  - 일본어 텍스트: `dc-(?:lang-ja )?dc-orig">([^<]+)</span>`
- **정렬**: 원본 숫자 파일명 기준 오름차순
- **파일명 규칙**: 특수문자(`\ / : * ? " < > |`) → `_`, 최대 100자, 공백 제거

## 에러 처리

- 파일 없음: 건너뛰고 계속
- 이름 중복: 덮어쓰기
- 기타 에러: 로그 출력 후 계속

## CSV 구조 예시

```
"앞면","[sound:12345.mp3] {{c1::<span class=""dc-lang-ja dc-orig"">食べる</span>::...}}"
```

→ `1_食べる.mp3`
