PT Krakatau POSCO — 設備現況 & Phase 2拡張計画

インドネシア・チレゴン(Cilegon)一貫製鉄所 — 現有設備のまとめと第2段階拡張動向
作成日 : 2026-06-09 根拠 : PTKR(Primetals Korea)メール (BonKyoung Koo) + 公開資料 宛先 : Markus / Tanaka-san

エグゼクティブサマリー

  • PT Krakatau POSCO(PTKP) はPOSCOとインドネシアのKrakatau Steel(PTKS)の合弁会社で、インドネシア初の高炉(BF)ベース一貫製鉄所。2013年12月稼働、粗鋼年300万トン(3.0 mtpy)規模。
  • 現有ライン:製銑(焼結・コークス・高炉#1) → 製鋼(BOF) → 連続鋳造 → 厚板(Plate Mill) 及び 熱延(Hot Rolling Plant / HSM)。製品はスラブ・厚板・熱延コイル(HRC)。
  • Phase 2拡張 の推進力が強化:2026年3月インドネシア大統領の訪韓、5月POSCOホールディングス張会長のインドネシア訪問によりPOSCO側の要求が伝達され、推進の合意形成。
  • 構想案:上工程はPhase 1とほぼ同一仕様(VE・一部中国製機材によるコスト削減)+ HRM 1.5 → 3.0 mtpy増強 + PLTCM 1基(約1.4〜1.8 mtpy) + CGL(CVGL)1基(0.55 mtpy)。
  • 実行可能性調査(FS)は2026年10月完了目標(インド/中国エンジニアリング会社またはPOSCO E&C)。完工目標2030年またはそれ以前 — Phase 1高炉改修(relining、2030年)との連携が重要変数。

3Phase 2拡張計画 (メール基盤)

推進背景 — 2026年3月インドネシア大統領の訪韓、2026年5月POSCOホールディングス張会長(Mr. Chang)のインドネシア訪問を通じて、POSCO側の要求事項がインドネシア側に十分に伝達された。その結果、Phase 2拡張推進の必要性について合意が形成されつつある。

3.1 上工程 (Upstream)

  • Phase 1と事実上同一仕様で推進(第2高炉など一貫ラインの複製方向)。
  • コスト削減(Cost Reduction)に注力:① バリューエンジニアリング(Value Engineering)適用、② 一部機材の中国製調達。
  • 完成時、粗鋼 3.0 → 6.0 mtpy 規模に拡大(第2高炉増設方向)。

3.2 下工程 (HRM & 冷延・めっき)

設備能力(構想)内容
HRM(熱延)増強 拡張 1.5 → 3.0 mtpy 既存の熱延能力を2倍に拡大する構想(現1.5 mtpyラインは設計上4.0 mtpyまで拡張可能)。
PLTCM 1基 新設 ≈1.4 〜 1.8 mtpy Pickling Line Tandem Cold Mill(酸洗-冷間圧延一体型)。HRCを冷延鋼板(CR)に圧延する中核ライン。
CVGL / CGL 1基 新設 0.55 mtpy 連続溶融亜鉛めっきライン(Continuous Galvanizing Line)。冷延鋼板を亜鉛めっき(GI/GA)し、自動車・家電・建材用の表面処理鋼板を生産。

※ PLTCM・CGLの新設は、PTKPがスラブ・厚板・熱延中心から冷延・めっき(高付加価値の平板材)まで拡張する方向を意味する。

4主要日程 / タイムライン

  • 2026.05 — POSCOホールディングス張会長のインドネシア訪問 → Phase 2推進の合意形成

  • 2030 — Phase 1高炉改修(BF Relining)予定
  • 2030またはそれ以前 — Phase 2完工目標(Phase 1改修日程との円滑な統合のため前倒しを検討)
日程上の重要変数 — Phase 1高炉が2030年に改修に入るため、Phase 2完工を2030年またはそれ以前に合わせ、生産空白を最小化し改修日程と円滑に統合する意図。すなわち2030年が事実上のデッドラインとして作用する可能性。

5Primetals視点 / 示唆

機会 (Opportunity)

  • 下工程新設(PLTCM・CGL) はPrimetalsの強み領域 — 冷延・めっきラインパッケージの受注可能性。
  • HRM 1.5→3.0 mtpy増強 時、圧延/周辺設備の改造・増設需要。
  • 上工程の第2高炉・製鋼増設に伴う付帯設備(ガス清浄・自動化・環境)の機会。
  • 2030デッドライン → FS段階(〜2026.10)からの早期関与が仕様先取りに有利。

留意/リスク

  • 上工程は「Phase 1同一仕様 + コスト削減」基調 → 中国製一部調達を明示。価格競争の激化。
  • FS実施がインド/中国エンジニアリング会社またはPOSCO E&C → 発注構図・仕様に影響。
  • 既存熱延はSMS group、厚板はMHMM供給 → 既設設備との連携・インターフェースを考慮。
  • まだ構想(検討)段階 — 能力・構成・日程すべて変動可能。
