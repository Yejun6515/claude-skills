## B. 숫자 입력칸 — 빈칸 (인터뷰에서 채움)

| 셀 | 의미 |
|---|---|
| V6 | **견적일** (date) — 이게 유효기일(K40=V6+45)을 자동결정하므로 반드시 채움 |
| S21/S24/S27 | 품목 수량 (Quantity) |
| V21/V24/V27 | 품목 금액 (Amount) — 단가 U는 V/S로 자동 |

> 입력 방식 주의: 이 양식은 **금액(V)과 수량(S)을 넣으면 단가(U)가 자동**. 단가만 알면 금액=단가×수량 계산해서 V에 입력.

## 검증 체크리스트 (draft 완성 후 자동 실행)

1. **마커 잔존 스캔**: `{{` 하나라도 남으면 → 안 채운 칸. 전부 0이어야 통과.
2. **이전 건 잔존**: Subject/NO.에 다른 프로젝트 코드(B025S079, K1H 등) 안 남았는지.
3. **견적일 V6 채움**: 비면 유효기일이 1900년대로 깨짐.
4. **결제조건 확인**: 자동통과 금지 — "이 고객 결제조건 맞나?"(특히 L/C) 반드시 질문.
5. **합계 재계산**: Σ(V21..)=V33, 각 U=V/S 일치(엑셀 재계산값).
6. **납기**: 단일 선적이면 SHIPMENT_MONTHS, 품목별 단계 납기면 월별표(Zhengrui 예시 G41:V46 블록) 추가했는지.
7. **통화 JPY** 표기 존재(V19).
