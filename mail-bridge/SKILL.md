# mail-bridge — PTJ 영업 메일 브릿지

예준님(PTJ 영업)이 하는 **고객↔일본 엔지니어 메일 중계**를 대신한다. 단순 번역이 아니라, 옵시디언 볼트를 읽어 **라우팅(누구에게/CC/어느 스레드)을 판단**하고 영업 맥락을 얹어 전달 초안을 만드는 브릿지.

## 핵심 원칙 (사용자 확정)
- **초안은 내용 위주.** 메일 발송은 예준님이 직접 하므로, 헤더 완비된 완성 메일이 아니라 **바로 쓸 내용**을 준다.
- **항상 두 벌:** ① 예준님 확인용 **한국어** (무엇을 어떻게 전달하는지 이해용) + ② 실제 발송용 **상대 언어**(JP 등, 복사해서 바로 사용).
- **PTKR(음대일 등)은 CC 원칙 제외.**
- 없는 사실은 지어내지 않는다. 불명확은 "미확인"으로 표시하고 surface한다(판단은 예준님).

## 모드 A — 전달 초안 만들기 (주 용도)

입력: 받은 메일 1통 (`.msg` 파일 / 폴더 / 붙여넣은 본문).

1. **읽기.** `.msg`면 `scripts\extract_msg.ps1`로 덤프 → Read.
   ```powershell
   powershell -NoProfile -ExecutionPolicy Bypass -File "<skillDir>\scripts\extract_msg.ps1" -FolderPath "<폴더>"
   ```
   제목/발신/수신/CC/본문/첨부를 파악. 답장 체인(RE:)은 과거 메일이 통째로 들어있으니 스레드 흐름을 시간순으로 푼다.
2. **패턴·매핑 로드.** 위 지식 소스 두 파일을 읽는다.
3. **라우팅.** 메일에서 프로젝트/설비/코드 키워드를 뽑아:
   - 먼저 `02_매핑\연락처-매핑.md`에서 프로젝트·담당자·CC 규칙 확인.
   - 볼트에서 관련 **프로젝트 MOC**(예: `PTJ Menu Order Code`)와 최근 Event 노트를 Grep/Read → **프로젝트 코드로 기존 상대측 스레드를 찾아 그 스레드에 RE로 회신**하는 게 기본.
   - **To** = 해당 안건 담당 엔지니어 / **CC** = 그 프로젝트 스레드·최근 방문 참가자 전원 / **PTKR 제외**.
   - 근거를 함께 제시(어느 노트/스레드에서 왔는지).
4. **초안 작성.** `패턴.md`의 작문 구조를 따른다. KR2JP 기본형:
   ```
   ① 커버노트 — 요청 핵심 1~2줄 + 영업 맥락(견적 일정·직전 미팅 등, 원문에 없어도 능동 추가)
   ② =<안건약칭> QTE= — 원문 전체 번역(충실, bullet 유지, 매핑표 고정 용어)
   ③ =UNQTE=
   ④ 맺음말 + 서명(キム)
   ```
   역방향(JP2KR/JP2CN)은 상대 언어로 쓰되 예준님 확인용 한국어를 반드시 병기.
5. **출력.** 아래 형식으로 채팅에 제시(내용 위주):
   - **회신 스레드**: (제목) / **To**: / **CC**: / **제외**: / **라우팅 근거**: 1줄
   - **【발송용 (일본어 등)】** — 복사용 본문
   - **【예준님 확인용 (한국어)】** — 같은 내용의 한국어
6. **자동 유지.** 메일에서 새 이메일/담당자/용어를 발견하면 `02_매핑\연락처-매핑.md`와 볼트 연락처(`20. Contacts`)에 반영(사용자 승인 없이 채워도 OK — 확정 선호).

## 모드 B — 케이스 학습 (규칙 성장)

입력: 케이스 폴더 `01_케이스\케이스NN_...\` (하위 `1.`=먼저 받음 / `2.`=뒤에 보냄).

1. 두 폴더의 `.msg`를 각각 추출해 **원문 ↔ 보낸 것**을 확보.
2. **대조 분석**: 라우팅(원문 To/CC vs 보낸 To/CC, 어느 스레드에 붙였나), 요약 밀도, 톤, 커버노트에 덧붙인 영업 맥락, 용어 매핑, 생략/강조.
3. 볼트에서 해당 프로젝트를 찾아 라우팅 근거를 검증.
4. **확인 포인트를 surface**(불명확·판단 필요 항목)하고 예준님 답을 받아,
5. 배운 것을 `00_가이드\패턴.md`(규칙·케이스 로그)와 `02_매핑\연락처-매핑.md`(연락처·CC·용어)에 반영, 볼트 연락처도 자동 채움.
