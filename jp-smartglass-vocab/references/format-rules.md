# 스마트 글라스 프롬프트 형식 규칙

## 기본 구조 (한 줄)

```
번호. 단어(よみがな), [단어2(よみがな)], 한자: 한자(음)=훈음 · 한자(음)=훈음, 설명
```

## 읽기(よみがな) 규칙

- **단어 단위**로 부착 — 개별 한자마다 붙이지 않음
  - ✅ `無一文(むいちもん)`, `成り上がる(なりあがる)`
  - ❌ `無(む)一(いち)文(もん)`, `成(な)り上(あ)がる`
- 히라가나만 있는 단어(ムダ, マネ 등)는 읽기 생략
- 문법 패턴(~てもムダ, ~なら 등)은 패턴 그대로 표기

## 한국 한자(훈음) 규칙

- 형식: `漢字(한국음)=훈음`
  - 예: `見(견)=볼견`, `返(반)=돌아올반`, `無(무)=없을무`
- 같은 단어 내 한자는 ` · `로 구분: `見(견)=볼견 · 返(반)=돌아올반`
- 다른 단어/항목 간은 ` / `로 구분

## 설명 유형별 가이드

### 일반 단어/복합어
- 한자 의미를 조합해 단어 뜻 도출
- 예: `商売=장사를(売) 상업적으로(商) 함`

### 관용구(慣用句)
- 반드시 어원 이미지 설명 포함 후 → 의미
- 예: `藪から棒に=덤불(藪)에서 막대기(棒)가 갑자기 튀어나오는 이미지→갑자기/느닷없이`
- 예: `知恵を絞る=지혜(知恵)를 꽉 짜낸다(絞)는 이미지→머리를 쥐어짜다`

### 속담(ことわざ)
- 속담 전체 의미 + 배경 간략 설명
- 예: `三つ子の魂百まで=세 살(三つ子) 때의 혼(魂)은 백 살(百)까지→세 살 버릇 여든까지`

### 수동형(~られる/~れる)
- 원형 → 수동형 변환 명시
- 예: `使う(부리다)→수동형 使われる(부려지다/이용당하다)`

### 문법 패턴(~てもムダ, ~ずに済む 등)
- 구성 요소 분해 + 합쳐진 의미
- 예: `~ずに済む: ~ず(~하지 않고)+済む(끝나다)→~하지 않아도 되다`

## 금지 사항

- ❌ 줄바꿈 — 반드시 한 줄
- ❌ 개별 한자에 읽기 부착
- ❌ 설명 없이 뜻만 나열
- ❌ 관용구에 어원 설명 생략
