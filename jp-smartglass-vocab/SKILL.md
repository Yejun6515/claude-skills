# JP 스마트 글라스 단어 설명 생성기

## 개요

일본 드라마 학습용 엑셀 B열의 단어/문법을 스마트 글라스에 투영할 수 있는 한 줄 설명으로 변환한다. 상세 형식 규칙은 `references/format-rules.md` 참조.

## 출력 형식 (한 줄, C열)

```
번호. 단어(よみがな), [단어2(よみがな)], 한자: 한자(음)=훈음 · 한자(음)=훈음, 어원/구조 설명→의미
```

**핵심 규칙:**
- 읽기(よみがな)는 개별 한자가 아닌 **단어 단위**로 부착: `無一文(むいちもん)` ✓ / `無(む)一(いち)文(もん)` ❌
- 한국 한자: `漢字(음)=훈음` 형식 (예: `見(견)=볼견 · 返(반)=돌아올반`)
- 관용구/복합어는 반드시 어원 이미지 설명 포함
- 문법 패턴(~てもムダ 등)은 구조 분해 후 의미 설명
- **모든 내용 한 줄** — 줄바꿈 없음

## 확정된 예시 3개

**32번 (복합어+동사):**
`32. 無一文(むいちもん)から ここまで 成り上がる(なりあがる), 한자: 無(무)=없을무 · 一(일)=한일 · 文(문)=돈문 / 成(성)=이룰성 · 上(상)=윗상 / 自分(자분)=스스로자·나눌분, 無一文=한 푼(一文)도 없음(無)=무일푼, 成り上がる=아래에서 위로(上) 이루어(成) 올라가다=출세하다`

**44번 (관용구):**
`44. 商売(しょうばい), 知恵(ちえ)を絞る(しぼる), 한자: 商(상)=장사상 · 売(매)=팔매 / 知(지)=알지 · 恵(혜)=은혜혜 / 絞(교)=짤교, 商売=장사를(売) 상업적으로(商) 함, 知恵を絞る=지혜(知恵)를 꽉 짜낸다(絞)는 이미지→머리를 쥐어짜다`

**73번 (동사+관용구+명사):**
`73. 見返す(みかえす), 一心(いっしん)で, 大学(だいがく), 한자: 見(견)=볼견 · 返(반)=돌아올반 / 一(일)=한일 · 心(심)=마음심 / 大(대)=클대 · 学(학)=배울학, 見返す=보고(見) 돌아간다(返)는 이미지→보란 듯이 갚아주다/해내다, 一心=하나의(一) 마음(心)=일념, 大学=큰(大) 배움(学)의 장`

## 워크플로

1. 엑셀 B열에서 단어/문법 항목 읽기
2. 각 항목별 C열 한 줄 생성 (형식 규칙 적용)
3. 엑셀 A열: `번호. 대표단어` 형식으로 업데이트
4. 결과를 새 엑셀에 저장
